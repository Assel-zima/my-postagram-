# Мой Постаграм

Небольшое React-приложение с лентой постов, созданное в рамках домашней работы.

## Возможности

- отображение списка постов
- рендер через `.map()`
- отдельные React-компоненты
- лайк через `useState`
- иконка из `react-icons`
- публикация на GitHub Pages

## Технологии

- React
- Vite
- JavaScript
- CSS
- react-icons
